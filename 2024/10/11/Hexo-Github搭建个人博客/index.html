XinWanna
2024-10-11
essays
  1. 前置安装

​ 由于之前我已经安装过Git和NodeJS，因此没有办法一步步截图了，贴两个安装链接供参考:https://blog.csdn.net/yaorongke/article/details/119085413 和 https://blog.csdn.net/yaorongke/article/details/119084295 .

  1. 创建Github项目

    • 点击GitHub右上角的头像，找到 Your repositories 并点击，之后点击 new 新建项目。
    • 项目的详细配置

    ​ ==注意：项目的名称必须是 Github用户名.github.io==

    • 获取SSH Key cd ~/. ssh 、ssh-keygen -t rsa -C "邮件地址(Github所绑定的邮件地址)"

    • 复制SSH

      从C盘中的用户目录下面找到 .ssh目录，找到目录下的id_rsa.pub文件，并复制里面的内容。

    • 配置SSH

    ​ 点击GitHub右上角的头像，找到 Setting,然后找到 SSH and GPG keys,创建一个新的SSH Key,并将复制的内容粘贴到里面。

    • 测试是否成功 ssh -T git@github.com # 注意邮箱地址不用改
    • 配置GitHub的用户名和邮箱

  2. Hexo安装

    • 安装Hexo

      在Git安装目录下面，创建一个文件夹（XinWanna），然后右键选择使用 Git Bash Here，之后输入npm install hexo-cli -g来安装Hexo

    • 查看Hexo版本 npx hexo -v

      ==注意：Hexo最新版本，必须在 hexo 命令前面加上 npx==

    • 创建Hexo项目 npm install --save hexo-deployer-git
    • 初始化 npx hexo init
