修改Hexo的默认主题

XinWanna
2024-10-11
Hexo+Github搭建个人博客

注意：当每次修改配置文件或相关文件时，需要使用 npx hexo g + npx hexo s / npx hexo d，才能在本地/网页查看到修改后的页面！！！

1.选择合适的主题并修改Hexo的配置文件

​ 从Hexo主题官网中，选择自己喜欢的主题，本文以cactus主题为例。

  • 下载主题 git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus

  • 修改Hexo中的主题 Hexo下载位置 -> _config.yml

    在_config.yml中，找到 theme，修改为所选择的主题（以cactus为例）。

2.修改Hexo的配置文件

​ 这部分是在Hexo下载位置的_config.yml文件里修改。

  • 修改博客的标题、副标题等等
key value
title 博客标题
subtitle 副标题
description 描述
keywords 关键字
author 作者
language 语言（汉语填 zh-CN）
timezone 时区（中国时区填 Asia/Shanghai）
  • 修改post_asset_folder为true

​ 此处主要是为了创建文章时，可以创建与文件名的同名文件夹，用来存放文章中所涉及的图片，以便于使用相对路径来显示图片。

3.个性化修改主题

​ 这部分是在Hexo下载位置\themes\cactus（即所下载的主题文件） 下的 _config.yml文件里修改。

  • 修改导航

​ 我将原有的导航修改成如下：home即为首页；essays即为所有文章的页面，这个的路径填写的是Hexo自带的文章汇总路径，即 /archives/；search即为搜索页面，这个是Hexo所提供的搜索功能的页面，可以看后文如何使用Hexo所提供的搜索功能。

​ 除了使用Hexo自带的，也可以自己创建页面npx hexo new page 页面名称，key可以自己起，value即为创建页面的文件夹路径（例如：Hexo下载位置\source\创建的文件夹）。

  • 修改社交媒体链接

​ 我只保留了github，然后添加了知乎。关于icon的选择，可以使用 fontawesome 中所提供的商标，网站中所提供的商标名即为icon的内容。

  • 首页显示标签
  • 修改首页所显示的文章个数

​ 默认显示5个，如果想要显示全部，注释掉 post_count: 5 ,然后加上 show_all_posts: true 这句话。

  • 修改版权年份
  • 修改logo设置

​ 具体说明看文件中的说明即可：

  • 修改网站图标

​ 虽然文件中有说明所使用的图标大小，但我并未严格设置，没有影响，可以正常显示。

  • 选择颜色主题

​ cactus支持的颜色主题有四种：dark, light, classic and white

4.添加评论

​ cactus支持两种，一个是 Disqus,一个是 Utterances，我使用的是Disqus。

  • 从 Disqus官网中为博客创建一个项目，一路跟着指引即可，并获取到项目的 shortname（shortname是创建完项目后，查看页面链接，链接中就包含了shortname,https://shortname.disqus.com/ 。当项目的名称为中文时，一定要根据链接来寻找 shortname，否则会出错）。

​ 配置完成后，就可以看到每篇文章下会有评论区。
